# InternSetu 🌉
### The AI-Powered Bridge Between Students & Internships

🔗 **Live Demo → [internsetu-production.up.railway.app](https://internsetu-production.up.railway.app)**

## What is InternSetu?
"Setu" means bridge in Hindi. InternSetu bridges the gap between talented students and companies hiring interns — using AI matching that actually understands both sides.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript (Single Page App)
- **Backend:** Python, Flask, MongoDB Atlas, JWT Auth
- **Deployed:** Railway.app

## Features
- 🤖 AI Skill Matching
- ⚡ One-Click Apply
- 📊 Application Tracker
- 🔖 Saved Roles
- 📈 Skill Gap Analysis
- 🔔 Notifications
- 🏢 For Companies

## Run Locally
```bash
git clone https://github.com/SubhashiniRanga2K5/InternSetu.git
cd InternSetu
pip install -r requirements.txt
python app.py
```
